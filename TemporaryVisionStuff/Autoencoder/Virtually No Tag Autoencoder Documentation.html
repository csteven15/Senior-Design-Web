Virtually No Tag Autoencoder Documentation¶

Introduction¶

This notebook is the development and documentation location for the abstract neural network to be used as an autoencoder for the Virtually No Tag project. That is, all code used to train the autoencoder will be in this document. The neural network will be exported for use elsewhere after having been trained here.

General Idea¶

The general idea with this is to have an autoencoder form a compact, database-storable representation of a picture of a license plate. This way, these N-dimensional vector representations can be stored and used to determine (via euclidean distance) if two pictures are of the same license plate.

def loadAllJPEGImages(path):
    print("Loading all JPEG images from disk...")
    imageList = sorted(listdir(path))

        if (not (imageName.endswith(".jpg") or imageName.endswith(".jpeg"))):

    print("Finished " + str(len(loadedImages[0])) + " loading!")
    output = np.concatenate(loadedImages, axis=0)
    return output

Model¶

The model to be used for this autoencoder will be relatively simple. It will consist of 192*96 sized input, since a standard plate is 12 inches by 6 inches, and thus has a 2:1 width-height ratio. This size was chosen by visual inspection, as it appears to allow sufficient quality to distinguish between all characters on the plate (Q vs O or 0, B vs 8, etc) and is otherwise relatively small (for faster training and usage of the network). Also, it is divisible by 2^4, or can be divided by 2 (downsampled in maxpooling) evenly 4 times.

# This method returns a distance metric between two encoded forms.
# Treats each channel as a 72-dimensional vector and computes
# euclidean distance between each channel. Then computes vector
# magnitude of the channel distances as the result.
def distances(encodedImages, index1, index2):
    channel0_1 = encodedImages[index1][:,:,0]
    channel1_1 = encodedImages[index1][:,:,1]
    channel2_1 = encodedImages[index1][:,:,2]
    
    channel0_2 = encodedImages[index2][:,:,0]
    channel1_2 = encodedImages[index2][:,:,1]
    channel2_2 = encodedImages[index2][:,:,2]
    
    channel0_mse = channel0_1 - channel0_2
    channel0_mse = channel0_mse * channel0_mse
    channel0_mse = np.sum(channel0_mse)
    channel0_mse = np.sqrt(channel0_mse)
    
    channel1_mse = channel1_1 - channel1_2
    channel1_mse = channel1_mse * channel1_mse
    channel1_mse = np.sum(channel1_mse)
    channel1_mse = np.sqrt(channel1_mse)
    
    channel2_mse = channel2_1 - channel2_2
    channel2_mse = channel2_mse * channel2_mse
    channel2_mse = np.sum(channel2_mse)
    channel2_mse = np.sqrt(channel2_mse)
    
    return np.sqrt(channel0_mse**2 + channel1_mse**2 + channel2_mse**2)
